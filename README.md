# My Dotfiles

## Installation

1. Install Xcode - Command Line Tools

2. Download Repo
```bash
git clone https://github.com/chrishelgert/dotfiles
```

3. Setup Git Credentials
Create your ``` ~/.gitconfig.local``` with your username and email.
*Example:*
```bash
[user]
    name = foo
    email = foo@example.io
```

4. Install dotfiles
```bash
install.sh
```

## Update

```bash
update.sh
```
