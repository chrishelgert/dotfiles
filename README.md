# My Dotfiles

## Installation

1. Download Repo
```bash
git clone https://github.com/chrishelgert/dotfiles
```

2. Setup Git Credentials
Create your ```bash ~/.gitconfig.local``` with your username and email.
*Example:*
```bash
[user]
    name = foo
    email = foo@example.io
```

3. Install dotfiles
```bash
install.sh
```

## Update
```bash
update.sh
```
